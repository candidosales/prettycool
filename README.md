# Pretty Cool Command Line Sound

> Pretty Cool! 

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g prettycool

## Running
Run command:

    prettycool

Or import in NodeJS script:

    import prettycool from 'prettycool';

    prettycool();
